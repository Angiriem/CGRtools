CGRTools
=========
CGR based reactions manipulations.


usage
=====
cgrtools --help


INSTALL
=======
    pip install -U git+https://github.com/stsouko/CGRtools.git@master#egg=CGRtools --process-dependency-links --allow-all-external

or

    pip install CGRTools
